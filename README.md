# Introduction to Python Programming

## About this project
I learnt Java programming and Object-Oriented Programming (OOP) principles. I learnt the basics of Java programming, including syntax, data types, control structures, and functions, as well as OOP concepts such as classes, objects, inheritance, polymorphism, encapsulation, and abstraction. The project includes hands-on sub-projects. I will have the skills to build robust and scalable software systems using Java and OOP principles.

## Prerequisites
The only prerequisites for this Java Programming and Object-Oriented Programming (OOP) project are a laptop, internet access, basic computer literacy skills, a problem-solving mindset and a determination to not give up.
